# Portfolio 2026

A personal portfolio built with React, TypeScript, Vite, and Tailwind CSS 4. It presents a home page, projects page, and contact page with a frosted-glass visual style, route-based navigation, and an animated Vanta background.

## Tech Stack

- React 19
- TypeScript 6
- Vite
- Tailwind CSS 4
- React Router
- lucide-react
- Vanta + Three.js

## Features

- Route-based navigation with breadcrumb-style navbar state
- Responsive home, projects, and contact pages
- Frosted-glass UI with themed gradients and cards
- Project cards with image previews, descriptions, tags, and external links
- Animated background powered by Vanta Waves

## Getting Started

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

Run linting:

```bash
npm run lint
```

## Project Structure

```text
src/
  components/   shared UI pieces like Navbar, Card, and ProjectCard
  css/          global theme variables and app styles
  data/         project data used on the projects page
  routes/       page-level route components
  types/        shared TypeScript types
  utils/        helper utilities
```

## Notes

- Project images live in `src/assets/images/`.
- The app uses CSS variables for theme colors and card surfaces.
- Route handling is currently done with `react-router`.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

You can also install [eslint-plugin-react-x](https://github.com/Rel1cx/eslint-react/tree/main/packages/plugins/eslint-plugin-react-x) and [eslint-plugin-react-dom](https://github.com/Rel1cx/eslint-react/tree/main/packages/plugins/eslint-plugin-react-dom) for React-specific lint rules:

```js
// eslint.config.js

import reactDom from "eslint-plugin-react-dom";

export default defineConfig([

      parserOptions: {
        project: ["./tsconfig.node.json", "./tsconfig.app.json"],
        tsconfigRootDir: import.meta.dirname,
      },
      // other options...
    },
  },
]);
```
